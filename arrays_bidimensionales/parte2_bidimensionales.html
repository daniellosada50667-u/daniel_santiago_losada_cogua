1 Sumatoria de Elementos FÁCIL

Desarrolle un algoritmo que recorra una matriz de enteros previamente inicializada y calcule la suma total de todos los valores almacenados en su interior.

Estrategia: Se declara una variable acumuladora fuera de los bucles. El ciclo anidado visita cada celda posicional sumando su contenido directamente al acumulador.
int[][] matriz = { {4, 8}, {3, 7} }; int sumaTotal = 0; for (int f = 0; f < matriz.length; f++) { for (int c = 0; c < matriz[f].length; c++) { sumaTotal += matriz[f][c]; } } System.out.println("Suma: " + sumaTotal); // Salida: 22
2 Extracción de la Diagonal Principal FÁCIL

En una matriz cuadrada (donde el número de filas es igual al número de columnas), la diagonal principal está compuesta por las celdas donde el índice de la fila coincide exactamente con el de la columna. Escriba el código para imprimir estos valores.

Optimización: En lugar de usar dos bucles y un condicional que filtre cuando f == c, es computacionalmente más eficiente utilizar un único bucle que acceda directamente a la coordenada idéntica de manera lineal.
int[][] cuadrada = { {10, 20, 30}, {40, 50, 60}, {70, 80, 90} }; // Recorrido optimizado con un único puntero de índice for (int i = 0; i < cuadrada.length; i++) { System.out.print(cuadrada[i][i] + " "); // Imprime: 10 50 90 }
3 Conteo de Elementos Condicionales MEDIO

Diseñe un programa que examine una matriz de datos numéricos y determine cuántos de esos elementos corresponden a números pares y cuántos a números impares.

Estrategia: Dentro del bucle interno se aplica el operador de residuo aritmético o módulo % 2 == 0 para discriminar la naturaleza del valor e incrementar el contador respectivo.
int[][] numeros = { {5, 12, 7}, {18, 3, 14} }; int pares = 0; int impares = 0; for (int f = 0; f < numeros.length; f++) { for (int c = 0; c < numeros[f].length; c++) { if (numeros[f][c] % 2 == 0) { pares++; } else { impares++; } } } System.out.println("Pares: " + pares + " | Impares: " + impares);
